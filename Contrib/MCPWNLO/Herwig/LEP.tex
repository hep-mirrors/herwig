\section[xxx]{T\scalebox{0.8}{HE}PEG\cite{ThePEG} Run Information}
Run name: \textbf{LEP}:\\
This run was generated using Herwig++~\cite{Bahr:2008pv} and the following models:
\begin{itemize}
\item 
\item In the running of $\alpha_{EM}$, the parametrization of H.~Buckhardt et al. was used. See \cite{KLEISSCERN9808v3pp129}.
\item The CLEO fits of \cite{Muramatsu:2002jp} and \cite{Kopp:2000gv} were used for the decays $D^0\to\bar{K}^0\pi^+\pi^-$ and $D^0\to K^-\pi^+\pi^0$.
\item The MAMBO algorithm of \cite{Kleiss:1991rn} was used for high multiplicity decays
\item The Shower evolution was perform using the algorithm suggested in \cite{Gieseke:2003rz}.
\item The decays of $\eta,\eta'\to\pi^+\pi^-\gamma$ were simulated using the matrix elements from \cite{Venugopal:1998fq,Holstein:2001bt}
\item The form factor model of \cite{Bauer:1986bm,Wirbel:1985ji} was used for either semi-leptonic or hadronic weak decays
\item The non-leptonic hyperon decays were simulated using the NonLeptonicHyperonDecayer class which implements the model of\cite{Borasoy:1999md}
\item The strong decays of the heavy baryons were simulated using the results of\cite{Ivanov:1999bk}.
\end{itemize}
